# Experiments

This folder contains runnable scripts for demonstrating system behavior.

## Purpose

- Showcase system correctness
- Demonstrate failure handling
- Provide reproducible scenarios

## Examples

- Partial commit simulation
- Idempotency verification

## How to Observe

When running experiments, observe:

- State changes before and after processing events
- Logs indicating invariant violations
- DLQ entries for rejected events
